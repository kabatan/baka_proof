# Source Map — v0.4.3

## Source principles inherited

- Full-first means no MVP / partial vertical slice completion.
- Base must remain domain-neutral.
- Raw solver outputs are not proof evidence.
- Proof-use requires validation, bridge/trust, and final verification.
- Geometry extraction is semantic extraction, not parsing.
- TraceCompiler and side-condition calculus are core research components, not decorative gates.

